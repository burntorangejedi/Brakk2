# Brakk2

A World of Warcraft addon for the Midnight expansion.

## Installation

1. Place the Brakk2 folder in your `World of Warcraft\_beta_\Interface\AddOns\` directory
2. Restart WoW or reload UI with `/reload`

## Usage

- `/brakk2` - Display help and version info
- `/brakk2 toggle` - Enable/disable the addon
- `/brakk2 status` - Show current status

## Features

- Basic addon framework with Ace3 libraries
- Saved variables support
- Event handling
- Slash commands

## Development

The addon uses the following structure:
- `Brakk2.toc` - Addon table of contents (metadata)
- `Core.lua` - Main addon logic

## Requirements

- World of Warcraft: Midnight (Interface 110002)
- Ace3 libraries (embedded or standalone)
